

# Community Service

## Version
|Module|Version|
|------|---|
|NodeJs|21.6.2|
|NestJs|10.3.2|
|Prisma|5.17.0|
|jwt|10.2.0|
|bcrypt|5.1.1|



## 설치

```bash
$ npm install
```

## 실행방법

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
